## Available Transformations

| Action | What It Does |
|---|---|
| **Rotate 90° CW** | Turns the image a quarter turn clockwise |
| **Rotate 90° CCW** | Turns the image a quarter turn counter-clockwise |
| **Rotate 180°** | Flips the image upside down |
| **Flip Horizontal** | Mirrors left-to-right (like a mirror image) |
| **Flip Vertical** | Mirrors top-to-bottom |

You can combine operations — rotate first, then flip, and the preview updates after each step.

## Common Use Cases

**Phone camera orientation:** Most phones embed orientation data (EXIF) in photos, but many apps and websites strip that data and display the raw pixels — sideways or upside down. Rotate 90° to fix it permanently.

**Mirroring for design:** Text overlays, watermarks, or logos sometimes need to be horizontally flipped for symmetry in a layout.

**Scanned documents:** Flatbed scanners placed face-down produce upside-down images. A 180° rotation fixes them instantly.

**Profile photos:** If a headshot came out slightly off-axis, a small rotation gets it right before uploading.

**Before/after comparisons:** Flip an image horizontally to create a mirrored version for side-by-side comparisons.

## Preview Before Downloading

The tool shows a live preview that updates as you rotate or flip. You can apply multiple transformations and see the cumulative result before downloading — no need to guess what 90° CCW looks like on your specific image.

---

## Output Quality

Rotation and flipping are lossless operations on the canvas — pixel values are rearranged, not recalculated. The output image quality matches the input.

For JPEG, the re-encoding step applies the same high quality setting used during the original compress. For PNG, it's fully lossless.

## Privacy: Runs in Your Browser

The Canvas API handles all rotation and flipping locally. Your image is never sent to a server — it stays on your device through the entire process.

## Two Ways to Scan

**Upload an image**

**Use your camera**
Point your device's camera at a QR code and it scans in real time — the same way a phone's native scanner works, but in any browser on any device.

## What It Decodes

The scanner identifies the content type automatically and formats it accordingly:

| QR Content | What you see |
|------------|-------------|
| URL | Clickable link |
| WiFi | Network name, password, encryption type |
| vCard / contact | Name, phone, email, address |
| Email | Address + pre-filled subject and body |
| Phone number | Tap-to-call link |
| SMS | Number + message text |
| Plain text | Raw text output |

## Common Use Cases

**QR codes received digitally**
When a QR code arrives as an image in an email, chat, or document, you can't point a phone at it. Upload the image file to scan it without needing a second device.

**Screenshots of QR codes**
Many apps generate QR codes you screenshot for later. Upload the screenshot and decode it instantly.

**Checking what a QR code contains before scanning**
Before scanning an unknown QR code, upload an image of it to see where it goes. Useful for verifying QR codes on printed materials, flyers, or shared images before you click anything.

**Restaurant menus and event tickets**
Digital menus and event QR codes are often sent as images. Upload them to get the link directly.

**WiFi QR codes**
Many routers and wifi setups generate a QR code for easy connection. Scan it to see the network name and password in plain text.

## Watermark Options

**Text content** — Any text: your name, website URL, `© Your Brand 2026`, `CONFIDENTIAL`, `DRAFT`, `SAMPLE`.

**Position** — Place the watermark in any corner (top-left, top-right, bottom-left, bottom-right) or centred. Bottom-right is the standard for photography.

**Opacity** — Slide from fully transparent to fully opaque. 30–50% opacity is the sweet spot — visible but not distracting.

**Font size** — Scale to fit your image. Larger images need larger text to be legible at standard viewing size.

**Colour** — White works on most images. Use a darker colour on light backgrounds or photos with bright skies.

## JPG vs PNG — Does It Matter?

Both formats work as input. The tool processes both equally.

For output:
- **JPG input → JPG output** — maintains the original format, smaller file size
- **PNG input → PNG output** — maintains transparency if present
- If you need the watermarked image with a transparent background, start with a PNG that has transparency
